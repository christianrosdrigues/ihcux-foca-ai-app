# FocaAí – Protótipo de Baixa Fidelidade

## Nome dos integrantes da equipe

CHRISTIAN DA SILVA RODRIGUES

## Conceito de Design

A interface do FocaAí foi pensada para reduzir a ansiedade do usuário usando uma navegação simples, poucos elementos por tela e bastante espaço em branco. O objetivo é mostrar apenas as informações essenciais, como o tempo restante e o progresso da árvore virtual.

O aplicativo evita excesso de notificações, mensagens de cobrança ou elementos visuais que possam gerar pressão. Em vez disso, usa uma linguagem calma e positiva, reforçando a ideia de que o usuário está ganhando tempo para a vida real ao se desconectar.

## Fluxo de Usuário

Fluxo para iniciar uma sessão de foco em grupo:

1. O usuário acessa a tela inicial do aplicativo.
2. Entra na área de grupo.
3. Escolhe ou cria uma sala de foco com amigos.
4. Define os aplicativos que serão bloqueados durante a sessão.
5. Escolhe o tempo de foco e o som ambiente.
6. Inicia a sessão de foco.
7. Durante a sessão, acompanha o tempo restante e o crescimento da árvore.
8. Ao finalizar, visualiza o relatório de conquistas e o ranking do grupo.

## Escolha de Componentes

Foram usados componentes simples para deixar a experiência mais calma e fácil de entender.

O cronômetro circular foi escolhido porque coloca o tempo restante como elemento principal da tela, facilitando a visualização rápida.

Os interruptores foram usados para bloquear aplicativos porque permitem ligar ou desligar cada opção de forma simples e direta.

Os cards foram usados para a escolha dos sons ambientes, pois deixam as opções mais visuais e fáceis de selecionar.

Os botões grandes ajudam o usuário a realizar as ações principais sem dificuldade, como iniciar o foco, continuar ou pausar.

O gráfico simples no relatório foi utilizado para mostrar o progresso do usuário sem causar excesso de informação.
